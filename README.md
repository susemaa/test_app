# Getting Started

Install dependencies:

```bash
npm ci
```
run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can also see skeleton [http://0.0.0.0:3000](http://0.0.0.0:3000) here.
